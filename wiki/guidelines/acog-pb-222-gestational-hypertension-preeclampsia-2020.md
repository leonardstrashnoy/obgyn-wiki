# ACOG Practice Bulletin 222: Gestational Hypertension and Preeclampsia (2020)

**Organization:** ACOG  
**Year:** 2020  
**Source:** See raw guideline file for URL

## Summary

Key recommendations for diagnosis, management, and prevention of progression from gestational hypertension to preeclampsia/eclampsia. Emphasizes timely delivery and magnesium sulfate for seizure prophylaxis.

## Key Recommendations (evidence-graded)

- **magnesium_sulfate** treats **preeclampsia** (Level 1A, ACOG PB 222)
- **labetalol** treats **preeclampsia** (Level 1B, ACOG PB 222)
- **nifedipine** treats **preeclampsia** (Level 1B, ACOG PB 222)
- **methyldopa** treats **preeclampsia** (Level 1B, ACOG PB 222)
- **magnesium_sulfate** prevents **eclampsia** (Level 1A, ACOG PB 222)
