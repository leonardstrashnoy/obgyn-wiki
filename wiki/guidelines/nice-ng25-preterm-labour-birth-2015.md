# NICE Guideline NG25: Preterm labour and birth (updated 2022)

**Organization:** NICE  
**Year:** 2015 (updated 2022)  
**Source:** See raw guideline file for URL

## Summary

Antenatal corticosteroids for fetal lung maturity and magnesium for neuroprotection when birth <30 weeks is imminent.

## Key Recommendations (evidence-graded)

- **betamethasone** prevents **preterm_labor** (Level 1A, NICE NG25)
- **magnesium_sulfate** neuroprotection **preterm_labor** (Level 1A, NICE NG25)
